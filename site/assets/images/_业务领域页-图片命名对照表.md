## 下载操作建议

1. 在 Midjourney 的 Organize / Create 历史中按上面关键词找到对应生成任务。
2. 按上表的 U1/U2/U3/U4 挑出推荐那张，点它做 Upscale（放大到单张高清）。
3. Upscale 完成后点图 → 右上角 **More / Download** → 保存原图。
4. 把下载到的文件按上表重命名，放入 `site/assets/images/` 目录。
5. 全部放好后用浏览器打开 `site/services.html` 验收。
