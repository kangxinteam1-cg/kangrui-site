# 康瑞六大业务方向 · Midjourney 提示词集

> **品牌视觉基因（House Style）**：深海军蓝 + 古铜金 + 暖金光，古典建筑构图，大理石与拉丝金属表面，戏剧性明暗对比，电影级写实，高端律所编辑视觉。
> 所有提示词都已带统一风格后缀，保证 6 张图放在一起协调。

---

## 用法

1. 复制下方代码块整段（包括 `--ar` 参数）粘到 Midjourney `/imagine`
2. 一次出 4 张 variation，按推荐 U1/U2/U3/U4 挑一张 Upscale
3. 下载后按"保存文件名"重命名，放到 `site/assets/images/` 或 `docs/assets/images/`
4. 每张提示词都给了 **A 方案 / B 方案** 两版构图，先跑 A，不满意再试 B

# 五、Midjourney 操作小贴士

- 跑出的 4 张里，**优先选光线最戏剧、明暗反差最强**的那张（与现有 area-01 风格一致）
- 如果觉得太暗，可在提示词加 `more luminous, brighter ambient, softer shadows`
- 如果觉得太亮俗气，可在提示词加 `darker, moodier, more cinematic shadows`
- 出现人物或文字水印的整批弃用，重跑
- 想保持 6 张 area 之间风格一致：把第一张满意的 area 图作 `--sref` 风格参考喂给后面 5 张
