01  /  确权争议

守住每一项权利
最关键的那条边界。

商标异议 / 无效 / 驳回复审 / 撤三　·　专利无效　·　版权登记争议　·　行政诉讼

一项权利能不能挺过未来的侵权战，往往早在它面对国家知识产权局、商标局和北京知识产权法院的那一刻就已经被决定。康瑞的口碑正是在确权战场上建立起来的——我们以高胜诉率守住客户最核心的权利边界，让后续的诉讼、许可与商业化都站在稳固的根基之上。

代表案例

驰名商标认定 + 无效宣告系列案 — BVLGARI / DEMAE ICCHO

通过针对性的事前知名度证据 + 抢注模式举证，先后为 BVLGARI、DEMAE ICCHO 等客户取得驰名商标认定，并成功无效争议商标，奠定行业内的标杆性结果。

02  /  诉讼维权

在法庭上以专业
而非声量取胜。

专利、商标、版权、商业秘密侵权诉讼　·　反不正当竞争　·　域名与企业名称争议

康瑞团队在最高人民法院、北京知识产权法院及各地专门法院，代理了大量复杂、疑难的侵权及不正当竞争案件——包括 SWAROVSKI、Lotus、Bureau Veritas、ORBOTECH、Bugaboo 等品牌的标志性诉讼。我们相信，真正的胜诉不是激烈的对抗，而是对事实、规则与节奏的精确掌握。

代表案例

驰名认定 + 跨境侵权诉讼 — SWAROVSKI · Lotus

SWAROVSKI 系统性举证驰名地位 + 被告对商标与企业名称的双重恶意模仿，判赔 21 万元；Lotus 包装案以历史驰名证据 + 比利时大使馆配合 + 平行无效被告外观专利的多线作战，判赔 40 万元。

03  /  调查取证

把模糊的事实，
变成可采信的证据。

背景调查　·　侵权调查　·　网络取证　·　公证购买　·　证据收集与固定

调查能力是康瑞区别于一般 IP 所的特色。从企业背景调查、商业秘密泄露追溯，到电商平台的网络监控、公证购买与现场监视，我们以多年沉淀的调查方法论，把细碎的线索锻造成完整的、可呈堂的证据链。

代表案例

半年跨省调查 + 控制性现场取证 — Bugaboo 商标刑事案

康瑞的半年深度调查锁定销售渠道、制造场所与两名核心人员，并通过控制性现场交易 + 组装行为固定"商标使用"证据，最终促成两名嫌疑人被捕、缓刑三年，对该地区婴儿车制造业起到显著震慑。

04  /  风控布局

让侵权
在发生前就被拦截。

行政查处　·　海关保护　·　展会保护　·　刑事打击　·　合规审查

真正高效的维权不发生在法庭——它发生在市场监管的投诉、海关的边境拦截、展会的现场监控，以及合规体系预先搭建的防御架构里。康瑞协助企业把"事后救济"前置为"事前布防"，把侵权拦在发生之前。

代表案例

市监行政查处 + 民事诉讼桥接 — ORBOTECH 软件案

以市监局的有利行政裁决为后续民事诉讼的关键证据，将行政与司法两条线打通，最终为客户获得 48 万元赔偿，最大化维权效率。

05  /  许可交易

让权利成为
看得见的商业价值。

谈判　·　转让　·　许可　·　登记备案　·　合同审核　·　IP 尽职调查

知识产权只有被"用起来"才真正产生价值。康瑞协助客户通过许可、转让、联合研发与质押融资等结构，撰写合同、完成尽调、谈判落地，把无形资产转化为可交易、可融资、可放大的核心竞争力。

06  /  战略咨询

在重大决策前，
提供更高一层的视角。

法律咨询　·　检索分析　·　IP 战略　·　涉外纠纷　·　公司与劳动法　·　常年顾问

康瑞以常年法律顾问的形式，与企业管理层、研发与品牌团队建立长期协作——覆盖产品发布前的检索分析、跨境业务的涉外纠纷协调、组织变革中的合规判断，以及面向高管、法务、研发与市场的定制化培训。我们把复杂的法律逻辑翻译成业务团队真正能用的判断工具。
